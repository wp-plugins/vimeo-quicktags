=== vimeo quicktag ===
Contributors: denzel chia
Tags: vimeo, quicktag
Tested up to: 2.7

== Description ==

Enable administrator to embed vimeo video into blog. Options available are the same as provided by vimeo.



== Installation ==


1. Upload `vimeo quicktag` to the `/wp-content/plugins/` directory and unzip
2. Activate the plugin through the 'Plugins' menu in WordPress
